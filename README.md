# Advanced-Web-based-Social-Networking-Engine

This is a web based social networking engine. The engine is a copmlete system which includes chat system,wall posts and home page system including time line.  The database structure is included also. THe server side is coded in php and the client side include a combination of ajax,jquery,javascript and html and css for web content display. The system runs on a mysql database using the mysqli-php extension.
